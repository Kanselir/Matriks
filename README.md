# Kalkulator Matriks

Kalkulator matriks online yang responsif dan mudah digunakan untuk berbagai operasi matriks.

## Fitur

- Penjumlahan matriks
- Pengurangan matriks
- Perkalian matriks
- Perkalian 2 matriks
- Penyelesaian persamaan matriks (mencari nilai x atau y)
- Transpose matriks
- Determinan matriks
- Invers matriks

## Cara Penggunaan

1. Buka file `index.html` di browser web Anda
2. Pilih operasi yang ingin dilakukan dari dropdown menu
3. Masukkan ukuran matriks yang diinginkan (baris dan kolom)
4. Isi nilai-nilai matriks
5. Klik tombol "Hitung" untuk melihat hasil
6. Gunakan tombol "Bersihkan" untuk mengosongkan semua input

## Catatan Penting

- Untuk operasi penjumlahan dan pengurangan, kedua matriks harus memiliki ukuran yang sama
- Untuk perkalian matriks, jumlah kolom matriks pertama harus sama dengan jumlah baris matriks kedua
- Untuk mencari invers matriks, matriks harus berbentuk persegi (jumlah baris sama dengan jumlah kolom)
- Untuk determinan, matriks harus berbentuk persegi
- Untuk penyelesaian persamaan, matriks koefisien harus berbentuk persegi

## Teknologi yang Digunakan

- HTML5
- CSS3
- JavaScript (Vanilla)

## Responsivitas

Kalkulator ini dirancang untuk bekerja dengan baik di berbagai ukuran layar, termasuk perangkat mobile.
